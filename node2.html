1 Download And Install

You can download all available versions of Candygram from SourceForge.net. The current stable release, as of this writing, is 1.0. You can also download this document in PDF format from the link above.

To install, uncompress the zip or tar file, cd into the directory, and run: 

python setup.py install

Windows users can just download the installer program and run it.
